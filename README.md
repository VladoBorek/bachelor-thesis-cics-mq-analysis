# bachelor-thesis-cics-mq-analysis
Analysis of transaction data from CICS and MQ systems

## Abstract
IBM mainframes generate large volumes of performance data through the System Management Facility (SMF) framework, yet extracting meaningful insights from this data remains challenging due to its complexity and limited tool accessibility. This thesis focuses on analyzing SMF records from the Customer Information Control Systems (CICS) and IBM Message Queueing (MQ) subsystems and proposes reusable guidelines for their interpretation, even without commercial analysis tools. The work combines theoretical background with practical data processing, including the development of a parser for CICS SMF 110 records and a structured approach to performance analysis. Concepts such as the transaction lifecycle and message flow are used to contextualize system behavior. Due to limitations in the available dataset, the MQ-related analysis is based primarily on theory. The thesis contributes a set of actionable approaches for SMF data analysis and aims to support both newcomers and those experienced with mainframes in understanding and analysing CICS and MQ system performance. méně

## Thesis Assignment
The aim of the bachelor thesis is to analyze data from transactional systems on the IBM z/OS platform and to create guidelines for their effective analysis. The student will become familiar with the mainframe environment, specifically with the IBM CICS Transaction Server for z/OS and IBM MQ for z/OS systems. Subsequently, he will analyze statistical data and performance parameters related to individual transactions, which the system generates in the form of SMF (IBM System Management Facilities) data. Part of the thesis will include a description of the transaction lifecycle and the development of guidelines for effective analysis of this data, with the goal of maximizing the impact on system operation and end users.
